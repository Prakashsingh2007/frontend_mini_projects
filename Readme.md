# Frontend Mini Projects 🚀

A collection of frontend projects built to strengthen my understanding of **HTML, CSS, JavaScript, and React**.  
These projects focus on improving core frontend development skills like UI design, DOM manipulation, state management, and component-based development.

## 📌 Projects Included

This repository contains three mini projects:

1. Calculator using HTML, CSS & JavaScript
2. Todo Application using Vanilla JavaScript
3. Todo Application using React

---

# 1. Calculator 🧮

## Overview

A simple calculator application built using **HTML, CSS, and JavaScript**.

The project focuses on understanding:
- HTML structure creation
- CSS styling and layout
- JavaScript event handling
- DOM manipulation
- Basic arithmetic operations

## Features

✅ Perform basic calculations  
✅ Addition, subtraction, multiplication, division  
✅ Clear/reset functionality  
✅ User-friendly interface  
✅ Button-based input handling

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

---

# 2. Todo Application (JavaScript) ✅

## Overview

A Todo application built using **Vanilla JavaScript** to practice working with the DOM and managing application data without any framework.

The project focuses on:
- Dynamic UI updates
- JavaScript logic building
- Event handling
- Data manipulation

## Features

✅ Add new tasks  
✅ Delete tasks  
✅ Mark tasks as completed  
✅ Update task status dynamically  
✅ Interactive user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

---

# 3. Todo Application (React) ⚛️

## Overview

A Todo application built using **React.js** to understand component-based frontend development.

This project is an upgraded version of the JavaScript Todo app, focusing on modern frontend practices.

## Concepts Practiced

- React components
- Props
- State management
- Event handling
- Conditional rendering
- Component communication
- React hooks

## Features

✅ Add tasks  
✅ Edit tasks  
✅ Delete tasks  
✅ Mark tasks as completed  
✅ Dynamic rendering using React state  
✅ Component-based structure

## Technologies Used

- React.js
- JavaScript (ES6+)
- CSS3
- Vite
